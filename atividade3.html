Qual é a diferença entre linguagem de programação Server Side e Client Side? Cite exemplos de cada tipo
Client Side

As linguagens cliente-side são aquelas cujo código é executado no navegador do usuário, do lado do cliente. Isso significa que o processamento ocorre no dispositivo do usuário, sem a necessidade de comunicação com o servidor.

Exemplo:
  • Javascript
Server Side

As linguagens server-side são aquelas cujo código é executado no servidor, ou seja, o servidor é capaz de entender e processar esse código. A resposta resultante é enviada para o navegador do usuário.

Exemplo:
  • Java
  • Ruby
  • PHP
  • Python
